Namaz Vakti API

Diyanetin sitesindeki verileri kullanarak namaz vakitleri verileri sunar. (Serves Praying Times data using the official website of the turkish directorate of religious affairs)
API ücretsiz ve erişime açıktır. Ücretsiz sunucu kullanıldığından ilk açılış 10-15 saniye sürebilir. İlk açılıştan sonra hızlı çalışır.
Kaynak Kodlar (GitHub)
namaz vakti uygulaması için oluşturulmuştur

Kullanım

Ülkelerin listesi: /countries
Şehirlerin/Eyaletlerin listesi: /cities?country=33
İlçelerin/Bölgelerin listesi: /regions?country=2&city=503
Vakit verileri: /data?region=9541
